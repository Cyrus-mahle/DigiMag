Git & Github

Git & Github are the most important tools in a work place or development environment or in project management, Almost all the companies now are using git and Github for project management, Git is a version control tool, A version control tool system allows a developer to keep track of changes to a project,document, or a source code while Github is a storage for project repositories, a repository contains your project in folders that are set up for version control tool.

Basic Git Commands:

  • git status
  • git commit
  • git reset
  • git log
  • git branch
  • git checkout
  • git merge
